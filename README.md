## Overview

This project is a Chatbot Flow Builder built using **React + Vite + React Flow**.
Users can create chatbot message flows by adding nodes and connecting them.

---

## Features

### 1. Text Node

* Drag and drop message nodes
* Multiple nodes supported
* Editable text via settings panel

### 2. Nodes Panel

* Contains available node types
* Currently supports Message Node
* Easily extendable for future node types

### 3. Edge Connections

* Connect nodes visually
* One outgoing edge allowed per source
* Multiple incoming edges allowed

### 4. Settings Panel

* Opens when a node is selected
* Allows editing message text

### 5. Save Validation

* Shows error if more than one node has no incoming edge
* Prevents invalid flow structure
